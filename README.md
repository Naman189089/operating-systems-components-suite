# operating-systems-components-suite

# Project Index

This repository serves as a centralized index for my systems and low-level programming projects.  
Each linked repository focuses on a specific core operating systems or concurrency concept.

---

## Projects

### Simple Loader
A minimal program loader that demonstrates the fundamentals of loading and executing binaries, including basic memory layout and control transfer.

Repository:  
https://github.com/Naman189089/OS-Simple-Loader

---

### Smart Loader
An extended version of the basic loader with additional intelligence, such as improved handling of program metadata and execution flow.

Repository:  
https://github.com/Naman189089/SimpleSmartLoader

---

### Simple Shell
A basic command-line shell implementation supporting command parsing and execution.  
Designed to illustrate process creation, execution, and basic input handling.

Repository:  
https://github.com/Naman189089/OS-Shell

---

### Simple Scheduler
A minimal scheduler implementation demonstrating core scheduling concepts such as task selection and context switching logic.

Repository:  
https://github.com/Naman189089/SimpleScheduler

---

### Simple Multithreader
A basic multithreading implementation illustrating thread creation, management, and concurrent execution.

Repository:  
https://github.com/Naman189089/OS-Multithreader
